# 송찬흠 포트폴리오 — 챗봇 지식베이스

## 개인 정보

이름은 송찬흠이고, 1992년 12월 12일생입니다. 인천 동구에 거주하고 있습니다. 2016년 11월부터 개발을 시작해 현재까지 약 10년차 개발자입니다.

## 소개

저는 프론트엔드부터 백엔드, 인프라까지 전 영역을 넘나드는 풀스택 개발자입니다. AI를 단순한 도구가 아닌 협업자로 활용하며, 팀에서 비어 있는 역할을 스스로 찾아 채우는 것을 강점으로 삼고 있습니다. 기술이 사용자에게 어떻게 느껴져야 하는지를 먼저 설계하고, 그에 맞는 기술을 선택합니다.

## 기술 스택

주력 언어는 TypeScript와 JavaScript입니다. 프론트엔드는 Next.js와 React를 중심으로, 애니메이션과 인터랙션에는 Framer Motion을 활용합니다. 스타일링은 Tailwind CSS를 사용합니다. 백엔드와 데이터베이스는 Supabase(PostgreSQL + pgvector)를 주로 사용하며, 인프라는 Vercel로 배포합니다. AI 관련 작업에는 Google Gemini API와 Claude API를 활용합니다.

## 경력

### 웹 프론트엔드 개발자, 수석 — (주)아이티센클로잇 (2024.08 ~ 현재)

아이티센글로벌 그룹사로, 아이티센글로벌은 매출액 8조 8,707억 원 규모의 회사입니다. 아이티센클로잇은 아이티센그룹의 그룹사로 고객사의 클라우드 전환, 구축 및 운영을 담당하는 총 인원 400명의 중견기업입니다. 새로운 패러다임의 Cloud MSP 사업 개발에 참여하고 있으며, 애자일 기반 R&D 팀 구성으로 CMP 개발 및 고객서비스포털 개발을 담당하였고, AI Agent 사업인 AgentGo2026 프로젝트에 개발 참여하고 있습니다.

### 웹 프론트엔드 개발자 — (주)알티캐스트 (2021.12 ~ 2024.07)

중견기업, 총 137명, 최고매출 466억 원 규모의 회사입니다. 카쉐어링 플랫폼 개발과 국내외 서비스 런칭에 참여했습니다. 사용자 편의성 증가 및 서비스 성장을 위한 지속적인 리팩토링을 진행했습니다.

### 웹 프론트엔드 개발자, 대리 — (주)에코플래그 (2020.11 ~ 2021.11)

스타트업에서 국가 수도 관련 SI 프로젝트를 진행했습니다. 개발리딩 참여, 애자일 방법론 도입, CI/CD 관리를 담당했으며, 프로젝트 솔루션 보일러플레이트를 개발했습니다.

### 웹/앱 개발자, 대리 — (주)카피앤페이스트 (2016.11 ~ 2020.11)

중소기업, 업력 19년차 회사에서 CMS, API, NDK, Socket Server 등 다양한 개발 경험을 쌓았습니다. 5000+ 어플리케이션 서비스 개발에 참여했습니다.

## 주요 프로젝트

이 포트폴리오 사이트 자체가 하나의 프로젝트입니다. CSS 3D 캐러셀, 커서 반응 조명 효과, AI 챗봇을 포함한 인터랙티브 포트폴리오로, Next.js, Framer Motion, Supabase, Google Gemini API를 활용해 만들었습니다. 경험을 먼저 설계하고 기술을 입히는 방식으로 개발했습니다.

블로그 페이지 리팩토링은 2024년 1인 프로젝트로, Notion API를 이용한 블로그 페이지 리팩토링입니다. 블로그 페이지 개발 ver.2로, Next14, React-Query, Tailwind CSS를 사용했습니다. 기존 md파일을 Notion으로 migration하고 Notion API를 이용한 게시글 호출 및 React-Query를 이용한 Data Cache 관리를 구현했습니다.

Daily Dev(Instagram Clone Coding)는 2024년 1인 프로젝트로, 개발 블로그, 게시글, docs 등 매일 포스트 추가 및 확인을 위한 북마크 페이지입니다. Next14, React-Query, Tailwind CSS를 사용했으며, 포트폴리오 root path 활용, Instagram migration, PWA를 활용한 모바일 APP 추가 기능 설정을 구현했습니다.

포트폴리오 페이지 개발은 2023년 1인 프로젝트로, 프로필 작성 및 개인 포트폴리오 작성을 위한 페이지 개발입니다. Next13, React-Query, Tailwind CSS, Framer-Motion을 사용했습니다. Parallax scroll 구현, 인터랙티브 웹 디자인 설계, 간편 제출용 이력서 페이지 구현, app route의 layout을 활용한 path별 기능 구현(/resume - 이력서, /main - 포트폴리오)을 포함합니다.

블로그 페이지 개발은 2023년 1인 프로젝트로, 기존 MERN Stack을 활용한 개발 블로그 페이지 리뉴얼입니다. Next13, Tailwind CSS를 사용했으며, Tailwind CSS를 이용한 Atomic Components 구조화 페이지 개발, MD 파일을 이용한 블로그 내용 작성을 구현했습니다.

## 연락처 및 채용 관련

현재 흥미로운 프로젝트나 팀을 찾고 있습니다. 풀스택 개발, AI 통합, 인터랙티브 UI 개발에 관심이 많습니다. 협업 제안이나 문의는 이 챗봇을 통해 남겨주시면 확인 후 연락드리겠습니다.

## 개발 철학

좋은 기술은 사용자가 의식하지 못할 때 완성된다고 생각합니다. 인터페이스의 무게, 반응의 속도, 전환의 리듬 같은 눈에 보이지 않는 디테일이 경험을 결정합니다. AI와 함께 사고하고, 함께 만들고, 그 과정 자체를 사용자에게 체감시키는 것이 목표입니다.
